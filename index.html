Compra comida que aún está buena y ayuda a reducir desperdicios

La idea de FasstSave Kevin es sencilla: mostrar comidas que un negocio ya preparo y venderlas con descuento antes de que se pierdan. Sirve como una simulacion de pedidos para restaurantes, panaderias o cafeterias.

Ver comidas Ver ofertas

Como funciona

1

El local registra lo que le queda disponible.

2

El cliente revisa el menu y escoge una opcion.

3

El pedido queda guardado en la tabla de control.

Resumen del sitio

El usuario puede revisar comidas, completar un pedido simulado y consultar la tabla de pedidos realizados.

La propuesta se enfoca en aprovechar alimentos disponibles antes de que se desperdicien.

Objetivo del proyecto

Este sitio muestra una propuesta para reducir el desperdicio de alimentos. Incluye inicio de sesion, registro, menu, ofertas, formulario de pedido y una tabla donde se ven los pedidos realizados.
